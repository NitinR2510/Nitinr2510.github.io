ICRYM

Posted: June 15, 2020

Project ICRYM ICRYM or I can read your mind is probably my first and still under development project. The aim of the project is to build a digital friend that everyone can connect to. Even though the original project was only one I had to make for my Database Management Systems course in my sophomore year, I have tried my best to create means to make the project more accessible. The one available through the link is still the old one, as the new one still is under development. At the moment, there is a model website and a telegram bot in test. I would be very happy if people can contribute ideas for the project. And I am even ready to rebuild it from scratch. Because this is certainly one of my most cherished one.

Just a Day: HTML Project

Posted: June 15, 2020

Just a Day The project aims at letting students relax. The project basically traces a day, a quiet and normal day in college. I hope that is it is peaceful to go through it. The project is yet built using HTML, CSS and basic JS alone. Optimised for desktop viewing only.
I was inspired by Lie Mao's simplistic web site that was captivating too. This theme is largely a carry-away of his. I am glad that platforms like github exist which further help in providing not just necessary open-source tools but also long-lost sense of inspiration to work forth and grow.
